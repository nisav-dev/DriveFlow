# DriveFlow — פלטפורמה להשכרת רכבים

מערכת תוכנה מלאה לניהול השכרת רכבים — פרויקט אקדמי.

## Stack טכנולוגי
- **Backend:** Java 17, Spring Boot 3.x, Spring MVC, Spring Data JPA, Spring Security
- **Frontend:** Thymeleaf, Bootstrap 5 RTL, JavaScript, Chart.js
- **Database:** PostgreSQL (או MySQL)
- **Build:** Maven
