# DriveFlow — מסכי מערכת

## מסכי לקוח (Customer Screens)

---

### SCR-C01: דף הבית (Home Page)
**URL:** `/`
**גישה:** כולם (מחוברים ולא מחוברים)

**תכולת המסך:**
- **Header:** לוגו DriveFlow, ניווט ראשי, כפתורי כניסה/הרשמה
- **Hero Section:**
  - כותרת גדולה: "מצאו את הרכב המושלם לנסיעה הבאה"
  - טופס חיפוש מוטמע עם שדות:
    - 📍 סניף איסוף (Dropdown)
    - 📅 תאריך ושעת איסוף
    - 📅 תאריך ושעת החזרה
    - 🔍 כפתור "חפש רכב"
- **קטגוריות רכב:** 4–6 קטגוריות עם תמונות (Mini, Economy, SUV, Luxury...)
- **יתרונות DriveFlow:** 3 כפתורי Feature (מחיר שקוף, ביטול חינם, שירות 24/7)
- **המלצות לקוחות:** Carousel עם ציטוטים
- **סניפים מובילים:** רשת עם מפה
- **Footer:** מידע על החברה, קישורים, יצירת קשר

### SCR-C02: תוצאות חיפוש (Search Results)
**URL:** `/search?pickupBranch=X&pickupDate=Y&returnDate=Z`
**גישה:** כולם

**תכולת המסך:**
- **סידורבר שמאל — פילטרים:**
  - סינון לפי קטגוריה (Checkboxes)
  - טווח מחיר (Slider)
  - תכונות רכב (מיזוג, אוטומטי, 4x4)
  - מיין לפי: מחיר, קטגוריה
- **תוכן מרכזי:**
  - סיכום חיפוש: "X רכבים זמינים · [סניף] · [תאריכים]"
  - רשימת כרטיסי רכב (Grid 2–3 עמודות):
    - תמונה ראשית
    - שם דגם + שנה
    - Badge קטגוריה (Economy / SUV / Luxury)
    - תכונות: 👥 X מושבים · ⚙️ אוטומטי/ידני · ❄️ מיזוג
    - מחיר ליום + סה"כ לתקופה מודגש
    - כפתור "הזמן" + "פרטים"

### SCR-C03: פרטי רכב (Vehicle Details)
**URL:** `/vehicles/{id}`
**גישה:** כולם

**תכולת המסך:**
- **גלריית תמונות** — Carousel + Thumbnails
- **פרטי הרכב:**
  - שם מלא (יצרן + דגם + שנה)
  - תיאור שיווקי
  - מפרט טכני בטבלה: סוג הנעה, תיבת הילוכים, מספר מושבים, מספר דלתות, נפח מטען
  - רשימת תכונות עם אייקונים
- **מחשבון מחיר:**
  - מחיר ליום
  - סה"כ לתקופה שנבחרה
  - כפתור "הזמן עכשיו"
- **מפת סניף:** Embedded map
- **רכבים דומים:** Carousel

### SCR-C04: תהליך הזמנה (Booking Wizard)
**URL:** `/booking/new`
**גישה:** לקוחות מחוברים בלבד

**שלבי ה-Wizard:**

**שלב 1 — תוספות (Extras):**
- סיכום ההזמנה בצד ימין (sticky)
- רשימת תוספות עם Checkbox + תיאור + מחיר ליום
- עדכון מחיר בזמן אמת

**שלב 2 — פרטי נסיעה ואישור:**
- שם מלא, מייל, טלפון (pre-filled מפרופיל)
- מספר רישיון נהיגה + תאריך פקיעה
- הסכמה לתנאים (Checkbox חובה)
- סיכום הזמנה מלא לפני תשלום

---

### SCR-C05: תשלום (Payment)
**URL:** `/booking/payment`
**גישה:** לקוחות עם הזמנה PENDING

**תכולת המסך:**
- **סיכום לתשלום** — Box ימין עם סה"כ
- **טופס תשלום Simulated:**
  - שם בעל הכרטיס
  - מספר כרטיס (format: **** **** **** ****)
  - תוקף + CVV
  - בחירת שיטה: Visa / MasterCard / PayPal
- **Security Badges:** SSL, Secure Payment
- כפתור "שלם [סה"כ] ₪ עכשיו"

---

### SCR-C06: אישור הזמנה (Booking Confirmation)
**URL:** `/booking/confirmation/{bookingNumber}`
**גישה:** לקוחות מחוברים

**תכולת המסך:**
- ✅ **אייקון הצלחה גדול** + "הזמנה אושרה!"
- **מספר הזמנה** מודגש: DRIVEFLOW-2026-00123
- **פרטי ההזמנה** בפורמט כרטיס:
  - 🚗 פרטי הרכב
  - 📅 תאריכים
  - 📍 סניף + כתובת + טלפון
  - 💳 סה"כ ששולם
  - 🎁 תוספות שנבחרו
- **כפתורים:** "הורד PDF" | "חזור לאזור האישי"
- הודעה: "מייל אישור נשלח ל-email@example.com"

### SCR-C08: היסטוריית הזמנות (Booking History)
**URL:** `/my-account/bookings`
**גישה:** לקוחות מחוברים

**תכולת המסך:**
- טבלה עם עמודות: מספר הזמנה, רכב, תאריכים, סניף, סה"כ, סטטוס, פעולות
- Pagination
- סינון לפי סטטוס/תאריך
- לחיצה על הזמנה → פרטים מלאים
- כפתור "בטל" להזמנות CONFIRMED

## מסכי מנהל (Admin Screens)

---

### SCR-A01: Dashboard ראשי
**URL:** `/admin/dashboard`
**גישה:** ADMIN, AGENT

**תכולת המסך:**
- **שורת KPIs (4 כרטיסים):**
  - 💰 הכנסות החודש
  - 📋 הזמנות פעילות
  - 🚗 אחוז ניצולת צי
  - 👥 לקוחות חדשים החודש
- **גרף הכנסות** — עמודות שבועיות (7 ימים)
- **טבלת הזמנות אחרונות** — 10 הזמנות אחרונות
- **סטטוס צי** — Donut Chart לפי סטטוס רכבים
- **התראות** — רכבים לתחזוקה, הזמנות ממתינות
