# DriveFlow — User Stories

---

## US-001: חיפוש רכב זמין

**כ-** לקוח רשום
**אני רוצה** לחפש רכבים זמינים לפי תאריך איסוף, תאריך החזרה ומיקום סניף
**כדי ש-** אוכל למצוא רכב מתאים לטיול שלי בזמן הנוח לי

**עדיפות:** גבוהה מאוד
**מודול:** Vehicle Search

---

## US-002: הזמנת רכב

**כ-** לקוח רשום
**אני רוצה** לבצע הזמנת רכב מקוונת עם בחירת תוספות (GPS, כיסא ילדים, ביטוח מורחב)
**כדי ש-** אוכל להבטיח את הרכב שאני צריך מראש ולקבל את כל הציוד שאני צריך

**עדיפות:** גבוהה מאוד
**מודול:** Booking

**כדי ש-** יהיה לי שליטה מלאה על הצי הזמין להשכרה

**עדיפות:** גבוהה מאוד
**מודול:** Vehicle Management
